# PDE

A python 3 library for solving initial and boundary value problems of some linear partial differential equations using finite-difference methods.

* Laplace
  * implicit central
* Parabolic
  * explicit central
  * explicit upwind
  * implicit central
  * implicit upwind
* Wave
  * explicit
  * implicit
